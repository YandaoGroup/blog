怎样使用

注册并登陆Discord

Discord官网：discord.com，注册登录过程不再赘述。

在这之后创建一个服务器。

接着，选择亲自创建

接下来，选择仅供我和我的朋友使用

最后，填写基本信息，点击创建就完成了Discord的准备工作。

邀请机器人

点击此处的链接邀请机器人

选择你刚才创建的服务器，点击继续

参照下图，点击授权

输入验证码，完毕后回到Discord上我们刚刚创建的服务器，可以看到机器人已经被添加进来了

开始使用

点击机器人的头像，可以查看其支持的命令

功能 作用
/swapid Swap ID - 上传被换脸的图片
/saveid Save ID - 储存要使用的人脸
/setid Set ID - 设置默认人脸，可用于右键快速换脸
/listid List ID - 列出所有已保存的人脸
/delid Del ID - 删除已保存的人脸

储存人脸

要换脸，首先要让机器人保存一张人脸，点击/saveid开始

idname即这个人脸的标识明，便于你区分不同人脸，image为人脸图像，建议上传没有佩戴眼镜、光线充足、清晰的正脸图像

当然，也可以像我一样不信邪，不过效果就没有保证了

如果你只上传了一张人脸图像，那么它就是默认的人脸，后面无需切换即可右键直接生图

如果上传了多张人脸，默认以最后一张为默认人脸，如需切换默认人脸，请使用/setid命令

一键换脸

接下来在聊天框中点击加号，上传一张图片，按回车键发送

接下来右键已发送的图片，选择App > INSwapper

然后看到机器人的回复了吗？我们现在只需要“坐和放宽——Microsoft”，耐心等候处理即可，一般几秒到十多秒也就好了
